🔍 Understanding Installation Issues

Windows 11 includes strict security settings. These settings often block third-party software installers. You might see a setup failure if the system permissions remain too restrictive. This guide provides clear steps to bypass these blocks and get your annotator running.

📥 Step-by-Step Installation Guide

Follow these steps to install the software on your PC. Perform each step in order to ensure success.

  1. Visit the official download page to select the latest version.
  2. Locate the setup file in your Downloads folder.
  3. Right-click the file and select "Run as administrator." This grants the application permission to modify local files.
  4. Confirm the Windows security prompt by selecting "Yes."
  5. Follow the setup wizard prompts. Accept the default directory settings unless you have a specific reason to change them.
  6. Click "Install" and wait for the status bar to reach completion.

🛠 Troubleshooting Common Errors

If the installation stops unexpectedly, check these common items:

  • Pending Windows Updates: Ensure your operating system is current. Windows 11 requires the latest patches for full compatibility. Check Settings, Windows Update, and "Check for updates."
  • Antivirus Interference: Sometimes security software flags the installer incorrectly. Disable your real-time protection for five minutes during the install process, then turn it back on.
  • Disk Space: Verify you have at least 500MB of free space on your primary hard drive.
  • Incomplete Downloads: If the installer produces a corruption error, delete the file and download it again. A partial download often causes setup failures.

📋 System Requirements

This software performs best on devices that meet the following points:

  • Operating System: Windows 10 or Windows 11.
  • Processor: 1.0 GHz or faster.
  • Memory: 4 GB RAM minimum.
  • Storage: 200 MB for the core application, plus extra space for your PDF documents.
